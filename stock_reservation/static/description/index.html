Sales Stock Reservation LAYAWAYS/ lay-by Odoo Apps

This Odoo apps is used for Product Booking/Stock Booking by the customer on Quotation. Whatever the reservation has been done, it will be displayed on product as reserved quantity also user can see which customer reserved this product and when , Once the reservation has been cancelled then reserved stock has been released. Displayed on product as reserved quantity and also user can see which customer reserved this product and when , Once the reservation has been cancelled then reserved stock has been released. When Product has been booked it will check the forcasted stock and existing reserved product based on that it will allow to reserved product. i.e Reservation quantity < Forcasted quantity - Reserved quantity then only it's allow to book the Reservation. Forcefully reservation option are also there,which can do forcefully reservation for product even there is no available stock for the reservation.

Features

Details about stock

User can see which customer reserved the product and when, once the reservation has been cancelled then reserved stock has been released.

Forecasted quantity

Check the forcasted stock and existing reserved product based on that it will allow to reserved the product. i.e Reservation quantity < Forcasted quantity - Reserved quantity then its only allow to book the reservation.

Forceful reservation.

Forcefully reservation option are also there,which can do forcefully reservation for product even there is no available stock for the reservation.

Reserved product availability

When product will be available (may be in few days/whatever time) we provide a option to check available of that products and if product is available, reservation state will became available.

Reserved Product Stock View

Here you can see reserved product's stock information as a Reserved Quantity and Reservation Line

Reserved Products Order View.

When trying to reserved any products and if products are available then state of the reservation will became available and it will be affected on reserved quantity of that product, and also reservation line will be shown on product.

Reservation / Sale Order View.

When customer reserved any products and if given products quantity is available then state of delivery order will be in available

When customer wants to reserve stock for any product and if products are not available then state of the reservation will became waiting availability and it will show check availability and force availability button to check and make it forcefully reserved. When product will be available (may be in few days/whatever time) we provide a option to check available of that products and if product is available, reservation state will became available.

Calculation for checking a reservation availability.

When Product has been booked it will check the forcasted stock and existing reserved product based on that it will allow to reserved the product. i.e Reservation quantity < Forcasted quantity - Reserved quantity then its only allow to book the reservation. If this satisfied then it's only allow to reserved

By clicking on force Availability button you can reserve products even if enough quantity is not available.
